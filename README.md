# rex_project
Rex Project Adalah Tools Untuk Menyebar Virus Via Redirect Link
Instalasi
Termux
1.pkg update && pkg upgrade
2.pkg install git
3.git clone https://github.com/JerkX/rex_project
4.cd rex_project
5.bash rex.sh

Linux
1.apt-get update -y && apt-get upgrade -y
2.apt-get install git
3.git clone https://github.com/JerkX/rex_project
4.cd rex_project
5.bash rex.sh
